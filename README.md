# Expense Tracker

A modern **Expense Tracker Web Application** built using **HTML, CSS, and JavaScript**.  
The application allows users to manage income and expenses, visualize financial data, and store transactions locally in the browser.

---

## Live Demo

🔗 https://expense-tracker-gsygalses-mergaurav4-gmailcoms-projects.vercel.app

---

## Features

- Add income and expense transactions
- Delete transactions
- Real-time balance calculation
- Transaction history with date tracking
- Expense visualization using **Chart.js**
- Dark mode toggle
- Mobile responsive user interface
- Data persistence using **LocalStorage**

---

## Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript (Vanilla JS)

**Libraries**
- Chart.js
